# Admin Cost Calculator

See exactly how much your broken systems are costing you.

## What This Does

An interactive calculator that shows:
- How much time you're wasting on admin
- How much revenue you're losing to poor follow-up
- Your total annual cost of operational chaos

**Perfect for:** Coaches, trainers, consultants, freelancers, service providers

## How to Use

1. Open `index.html` in a browser (or access via GitHub Pages link)
2. Answer 6 simple questions about your business
3. Get your personalized cost breakdown
4. Book a $100 audit to fix it

## The Math

For example:
- Spending 15 hours/week on admin at $150/hr = $117,000/year
- Losing 20% of leads to poor follow-up = another $48,000/year
- **Total: $165,000 in annual bleeding**

A working system costs $35-50/month and pays for itself in 3 weeks.

## Deploy

### GitHub Pages (Free)
1. Upload this file to a GitHub repo
2. Go to Settings → Pages → select main branch
3. Your calculator goes live at `https://yourusername.github.io/admin-cost-calculator/`

### Netlify (Free)
1. Drag and drop `index.html` to netlify.com
2. Get a live URL in 2 seconds

### Custom Domain
Upload to your own hosting and access at `yoursite.com/calculator`
